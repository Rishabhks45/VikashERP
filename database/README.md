# Database

Database scripts for the VikashERP workspace.

## Files

- `database_schema.sql` - full PostgreSQL schema blueprint for the ERP domain.
- `email_template_seed.sql` - seed data for email templates.
- `email_template_sms_seed.sql` - seed data for SMS templates.
- `email_templates_add_notification_type.sql` - migration/update script for notification type support.

Use new timestamped SQL files here for future schema/data changes.
